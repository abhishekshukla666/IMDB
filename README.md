🎬 IMDB Clone App

A simple IMDB-like iOS application built using Swift that allows users to browse movies, view details, ratings, posters, and basic movie information.

🚀 Features

* 📃 Movie listing screen
* 🔍 Search movies
* 🎥 Movie detail screen
* ⭐ Movie ratings
* 🖼️ Movie posters and thumbnails
* 📱 Clean and responsive UI
* 🌐 API integration for movie data
* ⚡ Async image loading

⸻

🛠️ Tech Stack

* Swift
* UIKit / SwiftUI
* MVVM Architecture
* URLSession
* Codable
* Auto Layout
* GCD / Async-Await

🔧 Installation

1. Clone the repository

git clone https://github.com/abhishekshukla666/imdb.git

2. Open the project in Xcode

open IMDB.xcodeproj

3. Run the project on simulator or device

🌍 API Used

This project uses movie data from:

* TMDB API
* OMDB API

You can generate your own API key from their official websites.
